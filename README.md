# 📊 FinPilot AI Dashboard

FinPilot is a full-stack financial intelligence dashboard designed to provide real-time insights into personal financial health using interactive analytics and AI-driven data visualization.

---

## 🚀 Features

- 📈 Real-time financial analytics dashboard
- 💰 Cash flow tracking (income vs expenses)
- 🏦 Multiple account management system
- 📊 Interactive charts and visual insights
- 🧠 AI-driven financial intelligence layer (future-ready architecture)
- 📱 Clean, modern UI built for usability

---

## 🧠 Tech Stack

- React (Frontend)
- Vite
- Recharts (Data Visualization)
- Node.js (Backend)
- FastAPI (API layer)
- MongoDB (Database)
